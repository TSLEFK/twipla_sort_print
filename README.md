# twipla_sort_print
https://twipla.jp/ の名簿をソートしたい。

Twiplaの名簿印刷機能は参加順になっているので、あいうえお順に並べ替えるボタンを追加するGoogle Chromeの拡張機能です。
公開はしていないので、自己責任でのインストールを実施してください。
